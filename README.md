# Custom HashMap in C++

A custom implementation of a hash map written in C++.

## Features
- Generic implementation using templates
- Custom hash function based on DJBX33A
- Collision handling with separate chaining
- Insert and update operations
- Index operator `[]`
- Iterator support
- Dynamic memory management
- Basic test cases in `main.cpp`

## Description
This project is a simple implementation of a hash map data structure in C++.  
It supports storing key-value pairs, handling collisions, iterating through elements, and accessing values with the index operator.

## Technologies
- C++
- Standard Library

## How to run
1. Open the project in your C++ environment or IDE
2. Compile `main.cpp`
3. Run the program

This project was created for educational purposes as a custom implementation of a common data structure.
